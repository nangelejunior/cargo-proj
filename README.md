## Projeto Rust com o Cargo

- gestor de projetos;
- gestor de ferramentas;
- ferramenta para executar tarefas administrativas.

```bash
cargo new proj
```

- Cargo.toml (arquivo principal de configuaração do projeto)
- src (programa principal)
  - main.rs

```bash
cargo build
```

```bash
cargo run
```

```bash
cargo fmt
```
